# Документация проекта "Монстры и Бумажки"

Документация игрового проекта, включающая:
- Концепцию игры
- Игровые механики
- Описание контента
- Технические спецификации

## Структура документации

- /sections - Разделы документации
- /css - Стили
- /assets - Ресурсы и изображения

## Версионность

Каждый раздел имеет свою версию и историю изменений.
